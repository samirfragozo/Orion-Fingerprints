The sample source builds in Visual Studio .NET 2005.

The sample source tests the LumiInOpAPI.

Be sure to set your working directory to ../../bin

Be sure there is a C:\temp folder on the PC.  If C:\temp
does not exist, sample code will not be able to save/load the files.  If you want to use a 
different folder to save/load files , change the DataPath constant to a different folder.
